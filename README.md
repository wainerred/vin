# 1win Landing

Одностраничный промо-лендинг. Статический сайт без сборки — открывается в любом браузере.

## Структура

```
vin/
├── index.html        # разметка страницы
├── css/
│   └── styles.css    # все стили
├── js/
│   └── main.js       # частицы и обработчики кнопок
└── README.md
```

## Запуск

Просто откройте `index.html` в браузере, либо поднимите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Что было сделано

Исходный монолитный `index.html` (с инлайновыми `<style>` и `<script>`) разбит на:

- **`css/styles.css`** — извлечённые стили, подключены через `<link>`.
- **`js/main.js`** — извлечённый скрипт, обёрнут в функции с `'use strict'`,
  инициализация по `DOMContentLoaded`, подключён с атрибутом `defer`.
- **`index.html`** — только семантическая разметка + `preconnect` к Google Fonts.
